# zsh-theme
My own ZSH theme based on [agnoster](https://gist.github.com/agnoster/3712874)

# Requirements

* powerline fonts
* ohmyzsh battery plugin (for battery status)

# Features

* SCM information (as in base agnoster theme)
* Kubernetes context and namespace (useful for `kubens`)
* non-zero exit code
* battery status in right prompt
* date in right prompt
